# Gateway

A WordPress plugin for collection registry and data management with React-based admin interfaces.

## Features

- Collection registry system with Eloquent ORM integration
- React-based admin and studio apps
- Grid support for tabular data display (TanStack Table)
- Form handling and schema generation
- Public API endpoints with filter handling
- Studio app with collection looping support

## Requirements

- WordPress 5.0+
- PHP 7.4+

## Installation

1. Upload the plugin files to `/wp-content/plugins/gateway/`
2. Run `composer install` to install dependencies
3. Activate the plugin through the WordPress admin panel
